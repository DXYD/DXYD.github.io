php的语法

PHP 脚本在服务器上执行，然后将纯 HTML 结果发送回浏览器。

基本的php语法

  • php脚本可以放在文档中的任何位置
  • 以<?php 开始 , 以 ？> 结束

php中的注释

  • 单行注释 //
  • 多行注释 /* */
  • 类似c语言

变量是用于存储信息的容器

php echo和print语句

echo 和print的区别

  • echo 可以输出一个或多个字符串
  • print 只能输出一个字符串，返回值总为1

echo输出的速度比print快，echo没有返回值，print有返回值1.

php echo语句

echo是一个语言结构，使用的时候可以不用家括号，也可以加上括号：echo或echo()

显示字符串

<?php
echo "<h3>php 牛逼！</h3>";
echo "hello<br>";
echo "my","blog","欢迎"
?>

显示变量

<?php
$txt1="学习 PHP";
$txt2="RUNOOB.COM";
$cars=array("Volvo","BMW","Toyota");
echo $txt1;
echo "<br>";
echo "在 $txt2 学习 PHP ";
echo "<br>";
echo "我车的品牌是 {$cars[0]}";
?>

php print语句

与echo 一样是一个语言结构，可以是用括号，也可以不适应括号：print()或print

php 数据类型

String（字符串）, Integer（整型）, Float（浮点型）, Boolean（布尔型）, Array（数组）, Object（对象）, NULL（空值）。

类似c语言的数据类型

数组可以在一个变量中存储多个值。可用php var_dump()函数返回数组的数据类型和值

<?php

var_dump($dong);

php null值

null值表示变量没有值。努力了是数据类型为null的值。
null值表示指明一个变量是否为空值。同样可用于控制和null值的区别。
可通过设置变量值为null来清空变量数据

<?php
$x="hello world";
$x=null;
var_dump($x);
?>

php比较类型

  • 松散比较：使用两个等号 == 比较，只比较值，不比较类型。
  • 严格比较：用三个等号 === 比较，除了比较值，也比较类型。

php常量

常量值被定义后，再脚本的的其他仁河地方都不能被改变。
一个常量由英文字母、下划线、和数字组成,但数字不能作为首字母出现。 (常量名不需要加 $ 修饰符)

设置php常量

使用define()函数

bool define(string $name,mixed $value [, bool $case_insensitive = false] )
  • name 必选参数，常量名称（标识符）
  • value 必选擦拭，常量的值
  • case_insensitive 可选参数,如果设置成TRUE，该常量则大小写不敏感，默认是大小写敏感的。

例如

<?php
define("DXYD","欢迎访问lirundong.top");
echo DXYD;      //输出"欢迎访问lirundong.top"
?>

常量是全局的
