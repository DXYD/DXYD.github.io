  • 先模拟浏览器发送requests请求——requests.get()

    requests发送请求

    1 requests.get(‘https://github.com/timeline.json’)# GET请求

2   requests.post(“http://httpbin.org/post”)  
# POST请求

3   requests.put(“http://httpbin.org/put”)     # PUT请求

4  requests.delete(“http://httpbin.org/delete”) # DELETE请求

5   requests.head(“http://httpbin.org/get”)     # HEAD请求

6   requests.options(“http://httpbin.org/get” ) # OPTIONS请求

  • response 收到服务器回应的网页（未解析）

    Response 对象的属性

    r.status_code

    http请求的返回状态，200表示连接成功，404表示连接失败

    r.text

    http响应内容的字符串形式，url对应的页面内容

    r.encoding

    从HTTP header中猜测的响应内容编码方式

    r.apparent_encoding

    从内容分析出的响应内容的编码方式（备选编码方式）

    r.content

    HTTP响应内容的二进制形式

    r.headers

    http响应内容的头部内容
