基于python的WordCloud词云的简单尝试

0 准备——安装python的第三方库如:

  • jieba

    中文分词

  • imageio

    导入图片

  • WordCloud

    词云工具

1 在互联网上找到所需图片

2 在Photoshop上编辑所需图片使得背景为纯白色

3 摘录txt格式的文本段落

  • 在互联网上自行寻找文字素材并保存

4 代码

import jieba
import wordcloud
import imageio
mk = imageio.imread("hello.png")

w = wordcloud.WordCloud(
                        scale=10,
                        background_color="white",
                        font_path="msyh.ttc",
                        mask=mk)

f = open("WordCloud.txt",encoding="utf-8")
text = f.read()
_text = jieba.lcut(text)
string = "".join(_text)
f.close()
w.generate(string)
w.to_file("word.png")
